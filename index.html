📡 How to connect real social data
Each platform requires its own API connection. Here's how:
LinkedIn: LinkedIn Marketing API — requires a Company Page Admin account + app approval at developer.linkedin.com
Facebook & Instagram: Meta Graph API — connect via business.facebook.com → Settings → Instagram/Facebook Insights
YouTube: YouTube Data API v3 — enable at console.cloud.google.com (same project as GA4) → add to anna-view-ai-bot service account
TikTok: TikTok Business API — apply at developers.tiktok.com/products/business-api (requires business account)
